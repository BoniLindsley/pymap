pymap.concurrent¶

Implements some concurrency utilities used by pymap. Each has an implementation using asyncio and :mod`threading` primitives.

class pymap.concurrent.ReadWriteLock¶

Read-write lock.

classmethod for_asyncio()¶

Return a read-write lock for asyncio.

Return type:ReadWriteLock
classmethod for_threading()¶

Return a read-write lock for threading.

Return type:ReadWriteLock
subsystem¶

The sub-system the read-write lock was created for, 'asyncio' or 'threading'.

Return type:str
read_lock()¶

Acquires a read-lock, blocking until any write-locks are released.

Return type:Abstractasynccontextmanager[None]
write_lock()¶

Acquires a write-lock, blocking until all read- or write-locks are released.

Return type:Abstractasynccontextmanager[None]
