class pymap.backend.redis.Config(args, *, address, select, prefix, users_hash, users_key, users_json, **extra)¶

Bases: pymap.config.IMAPConfig

The config implementation for the redis backend.

Parameters

  • args (Namespace) – The command-line arguments.

  • address (str) – The redis server address.

  • select (Optional[int]) – The redis database for mail data.

  • prefix (str) – The prefix for mail data keys.

  • users_hash (Optional[str]) – The hash key for user lookup.

  • users_key (str) – The user lookup key template.

  • users_json (bool) – True if the user lookup value contains JSON.

property address¶

The redis server address. Defaults to a connection to localhost.

See also

aioredis.create_connection()

Return type

str

property select¶

The redis database for mail data. If given, the SELECT command is called after successful user lookup.

Return type

Optional[int]

property prefix¶

The prefix for mail data keys. This prefix does not apply to user lookup, e.g. users_hash or users_key.

Return type

str

property users_hash¶

The name of the hash where keys are looked up for user login. If this value is None, which is the default, keys are looked up in the full redis database.

Return type

Optional[str]

property users_key¶

The key template for looking up user login. The login name (authcid) is substituted in the template in place of {name} using str.format(). By default, the template is {name} so the key will be the unaltered login name.

Return type

str

property users_json¶

True if the value from the user lookup key contains a JSON object with a "password" attribute, instead of simply containing the password value encoded with UTF-8.

Return type

bool

property namespace_hash¶

The name of a hash used to override user namespace prefixes – not needed for normal operation.

Typically, when a user first logs in it is assigned a random hex string used to namespace all the keys related to its mailbox data. That means another user name can be assigned the same string, to share the mailbox or rename it.

Return type

str
