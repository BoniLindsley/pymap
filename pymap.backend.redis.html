class pymap.backend.redis.Config(args, *, address, users_hash, users_key, users_json, **extra)¶

Bases: pymap.config.IMAPConfig

The config implementation for the redis backend.

Parameters:
  • args (Namespace) – The command-line arguments.
  • address (str) – The redis server address.
  • users_hash (Optional[str]) – The hash key for user lookup.
  • users_key (str) – The user lookup key template.
  • users_json (bool) – True if the user lookup value contains JSON.
address¶

The redis server address. Defaults to a connection to localhost.

See also

aioredis.create_connection()

Return type:str
users_hash¶

The name of the hash where keys are looked up for user login. If this value is None, which is the default, keys are looked up in the full redis database.

Return type:Optional[str]
users_key¶

The key template for looking up user login. The login name (authcid) is substituted in the template in place of {name} using str.format(). By default, the template is {name} so the key will be the unaltered login name.

Return type:str
users_json¶

True if the value from the user lookup key contains a JSON object with a "password" attribute, instead of simply containing the password value encoded with UTF-8.

Return type:bool
