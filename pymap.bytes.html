class pymap.bytes.BytesFormat(how)¶

Bases: object

Helper utility for performing formatting operations that produce bytestrings. While similar to the builtin formatting and join operations, this class intends to provide cleaner typing.

Parameters:how (bytes) – The formatting string or join delimiter to use.
format(data)¶

String interpolation into the format string.

Parameters:data (Iterable[Union[Bytestring, memoryview, Supportsbytes, int]]) – The data interpolated into the format string.

Examples

BytesFormat(b'%b, %b!') % (b'Hello', b'World')
Return type:bytes
