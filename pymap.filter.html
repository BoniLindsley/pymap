pymap.filter¶

class pymap.filter.EntryPointFilterSet(entry_point, value_type, *, group='pymap.filter')¶

Bases: pymap.interfaces.filter.FilterSetInterface

Base class for filter set implementations that use a filter compiler declared in the pymap.filter entry point. The declared entry points must sub-class FilterCompiler.

Parameters:
  • entry_point (str) – The entry point name.
  • group (str) – The entry point group.
