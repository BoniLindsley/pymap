pymap.plugin¶

pymap.plugin.PluginT¶

The plugin type variable.

alias of TypeVar(‘PluginT’)

class pymap.plugin.Plugin(group=None)¶

Plugin system, typically loaded from pkg_resources entry points.

>>> example: Plugin[Type[Example]] = Plugin('plugins.example')
>>> example.add('two', ExampleTwo)
>>> example.registered
{'one': ExampleOne, 'two': ExampleTwo}

Note

Plugins registered from group entry points are lazy-loaded. This is to avoid cyclic imports.

Parameters

group (Optional[str]) – A entry point group to load.

property registered¶

A mapping of the registered plugins, keyed by name.
