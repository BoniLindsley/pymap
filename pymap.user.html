pymap.user¶

class pymap.user.UserMetadata(config, password, *, params=None)¶

Contains user metadata including the password or hash.

Parameters

  • config (IMAPConfig) – The configuration object.

  • password (Optional[str]) – The password string or hash digest.

  • params (Optional[Mapping[str, str]]) – Additional metadata parameters associated with the user.

classmethod from_dict(config, data)¶

Build a new UserMetadata from a dictionary containing the password information. Fields must either be str() or int(), bytestrings should be hex-encoded.
